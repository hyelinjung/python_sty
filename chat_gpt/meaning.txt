[DataFrame은]
파이썬의 DataFrame은 주로 pandas 라이브러리에서 제공하는 2차원 데이터 구조로, 엑셀 스프레드시트나
SQL 테이블처럼 행과 열로 구성된 데이터를 다룰 수 있게 해준다
[주요기능]
대규모 데이터를 효율적으로 처리할 수잇는 다양한 기능을 제공
CSV,Excel,SQL 데이터베이스등 다양한 데이터 형식을 읽고 쓸수 있다
데이터 정제,병환,결합 및 시각화에 필요한 다양한 메서드를 제공

[seaborn은]
matplotlib보다 더 직관적이고 간결한 인터페이스를 제공하며 통계적으로 분석 결과를 시각화하는데
유용한 기능을 가지고 있음

[비교문의 is]-> if b is None
is연산자는 객체의 메모리주가 같은지 비교한다. None은 파이썬에서 유일한 객체이므로
none을 사용하는 것이 == 보다 정확하다
[is와 ==차이점]
is 객체의 메모리 주소를 비교
== 객체의 값이 같은지를 비교
